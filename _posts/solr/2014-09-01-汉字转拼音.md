## pinyin4j

* 支持中文简体/繁体
* 支持多音字
* 支持大小写

#### 下载

<http://sourceforge.net/projects/pinyin4j/files/>

#### 目录

* lib - jar包
* src - 源码
* doc - api文档

#### 初步了解

    java -jar lib/pinyin4j-2.5.0.jar 

<http://blog.csdn.net/hfhwfw/article/details/6030816>
